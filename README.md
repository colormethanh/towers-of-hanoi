## How To Play Game in the console

NOTE: This game used the index starting from 0. So the first peg is index 0, and last peg is index 2.

### To move:

`GAME.hndleMove(from-index, to-index)`

### To check if winner:

`GAME.checkWinner()`

### To reset game to beginning:

`GAME.resetGame()`
